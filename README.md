## Description

Api build using Nestjs and Sqlite

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# production mode
$ npm run start:prod
```

## Running the app on Docker

```bash
# Use the command on git bash 
$ docker run -p 3000:3000 -it backend-bankme:bankme
```
## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
